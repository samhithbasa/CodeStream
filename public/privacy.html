Privacy Policy

Last Updated: February 12, 2026

1. Information We Collect

We collect information you provide directly when you register, including:

  • Email address and password (hashed with bcrypt)
  • Google account information (if using Google OAuth): name, email, profile picture
  • Code snippets and projects you create and save
  • Frontend projects including HTML, CSS, JavaScript files and uploaded assets
  • Support tickets and contact form submissions

2. How We Use Your Information

We use your information to:

  • Provide and maintain our code editor and frontend playground services
  • Authenticate users via JWT tokens and Google OAuth
  • Store and retrieve your saved code and projects
  • Send OTP verification emails and password reset links
  • Respond to support tickets and contact inquiries
  • Improve and personalize your coding experience
  • Ensure the security of our service

3. Data Storage and Security

We implement appropriate security measures to protect your data:

  • Passwords are hashed using bcrypt with salt rounds
  • JWT tokens for secure authentication (7-day expiry)
  • Data stored securely in MongoDB Atlas
  • Rate limiting to prevent brute force attacks
  • HTTPS encryption for data transmission

4. Third-Party Services

We use the following third-party services:

  • MongoDB Atlas: For secure database storage
  • Google OAuth 2.0: For authentication (optional)
  • Gmail API / Brevo / Resend: For sending emails (OTP, notifications)
  • Render: For cloud hosting and deployment

These services have their own privacy policies governing their use of your information.

5. Code Sharing and Visibility

When you share code using our sharing feature:

  • Shared code becomes publicly accessible via the generated link
  • Anyone with the link can view your shared code
  • You retain ownership of all code you create
  • Private saved code is only accessible to you when logged in

6. Cookies and Tokens

We use JWT tokens stored in your browser to maintain your login session. These tokens expire after 7 days for security purposes.

7. Data Retention

We retain your account information and saved code until you delete your account. You can request account deletion at any time, which will permanently remove all your data.

8. Changes to This Policy

We may update this policy occasionally. We'll notify you of significant changes by email or through our service.
