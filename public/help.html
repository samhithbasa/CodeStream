How to Create an Account

  1. Click on the "Sign Up" button in the top right corner
  2. Enter your email address and choose a strong password
  3. Verify your email address by entering the OTP sent to your inbox
  4. Complete your profile setup (optional)
  5. Start coding immediately!

Note: If you don't receive the OTP, check your spam folder or request a new one.

Sign in with Google

For quick and easy access, you can sign in using your Google account:

  1. Click "Sign in with Google" on the login page
  2. Select your Google account
  3. Authorize CodeStream IDE to access your basic profile information
  4. You'll be automatically logged in and redirected to the editor

Note: We only access your name, email, and profile picture. We never access your Google Drive or other services.

Your First Program

Let's create a simple "Hello World" program:

    printf("Hello, World!");
    return 0;
}
  1. Select C language from the icons
  2. Type or paste the code above
  3. Click "Run" to execute
  4. See the output in the bottom panel

Saving and Organizing Your Code

To save your code:

  1. Click the "Save" button
  2. Enter a descriptive name for your code snippet
  3. Click "Confirm"

To access saved codes:

  1. Click the "Saved Codes" button
  2. Browse your list of saved programs
  3. Click any item to load it into the editor

Note: You must be logged in to save code.

Sharing Your Projects

Share your code with others in 3 easy steps:

  1. Save your code if you haven't already
  2. Click the "Share" button next to the saved code
  3. Copy the generated link and send it to anyone

Note: Shared links are public - anyone with the link can view your code.

Using the Integrated Terminal

CodeStream IDE includes a WebSocket-based terminal for interactive commands:

  • Access the terminal from the editor interface
  • Run shell commands and scripts
  • Debug your programs interactively
  • View real-time output

Note: Terminal access may be limited for security reasons.

Getting Started with Frontend Playground

The Frontend Playground lets you build web applications:

  1. Navigate to the Frontend Editor from the landing page

  4. Save and share your projects

Working with Multiple Files

Organize your frontend projects with multiple files:

  • Create separate HTML, CSS, and JavaScript files
  • Link files together using standard web practices
  • Manage file structure for complex projects
  • All files are saved together as a project

Uploading and Managing Assets

Add images and other assets to your web projects:

  1. Click the "Upload Asset" button
  2. Select your image or file (max 50MB)
  3. Reference the asset in your HTML/CSS code
  4. Assets are stored with your project

Common Error Messages

"Compilation Error"

This means your code has syntax errors. Check for:

  • Missing semicolons
  • Unmatched brackets
  • Typos in keywords

"Timeout Error"

Your code took too long to execute. Try:

  • Optimizing loops
  • Removing infinite loops
  • Reducing complex calculations

Code Not Running Properly

If your code runs but doesn't produce expected results:

  1. Check your logic for errors
  2. Add print statements to debug
  3. Compare with working examples
  4. Try smaller test cases

For language-specific help, refer to documentation:

C Documentation C++ Documentation Java Documentation Python Documentation JavaScript Documentation

Login Issues

Having trouble logging in? Try these solutions:

Forgot Password

  1. Go to the login page
  2. Click "Forgot Password"
  3. Enter your registered email
  4. Check your inbox for reset instructions

Google OAuth Issues

If Google sign-in isn't working:

  • Clear your browser cache and cookies
  • Try a different browser
  • Ensure pop-ups are enabled
  • Check your Google account is active

Account Locked

After multiple failed attempts, your account locks temporarily for security.

Wait 15 minutes or contact support at samhithbasa12@gmail.com.

Updating Your Profile

To update your account information:

  1. Click your profile picture in the top right
  2. Select "Account Settings"
  3. Update your email, name, or other details
  4. Click "Save Changes"

Note: Changing your email requires verification.

Changing Your Password

You can change your password two ways:

When Logged In

  1. Go to Account Settings
  2. Select "Security"
  3. Enter current and new password
  4. Confirm changes

Password Reset

  1. From login page, click "Forgot Password"
  2. Follow emailed instructions
  3. Create a new strong password (minimum 8 characters)

Deleting Your Account

To permanently delete your account:

  1. Go to Account Settings
  2. Scroll to "Delete Account"
  3. Read the warning carefully
  4. Enter your password to confirm
  5. Click "Permanently Delete"
warning

This action cannot be undone. All your saved code, projects, and data will be permanently deleted.
